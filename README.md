# owlcode

## Install

Windows

```
$ git clone https://github.com/tadashi-aikawa/owlcode.git %HOMEPATH%\.vscode\extensions\owlcode
$ cd %HOMEPATH%\.vscode\extensions\owlcode
$ npm i
$ npm run compile
```
