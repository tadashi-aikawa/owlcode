# owlcode

## Install (Windows)

```
$ curl "https://raw.githubusercontent.com/tadashi-aikawa/owlcode/master/install.bat" -o install.bat
$ install.bat
```

## Update (Windows)

```
$ curl "https://raw.githubusercontent.com/tadashi-aikawa/owlcode/master/update.bat" -o update.bat
$ update.bat
```
